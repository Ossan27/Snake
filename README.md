# Snake

A classic Snake game project built with PlatformIO.

## Overview
This project implements the Snake game, where a player controls a snake that grows by eating food while avoiding collisions with walls and itself.

## Features
- Snake movement and growth mechanics
- Food generation and collision detection
- Game over conditions
- Score tracking

## Getting Started

### Prerequisites
- PlatformIO
- Compatible microcontroller or development board

### Installation
1. Clone or open this project in PlatformIO
2. Install dependencies: `pio lib install`
3. Build the project: `pio run`
4. Upload to your device: `pio run --target upload`

## Usage
- Control the snake direction using input controls
- Eat food to grow and increase your score
- Avoid hitting walls and yourself
